Nourish — Daily Calorie Planner 🌿

A simple web application that helps users calculate and manage their daily calorie intake based on personal profile details and selected foods.

The project guides users through three steps:

Enter personal profile details

Select food items

View daily calorie summary

This tool demonstrates how basic web technologies can be used to build a practical health-related application.

Features

User profile input (Age, Gender, etc.)

Daily calorie calculation

Food selection interface

Step-by-step navigation system

Final calorie summary

Clean and responsive UI

Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling and layout

JavaScript – Logic and calorie calculations

One thing you should consider (important)

Right now your project is basically a calculator with UI. That’s fine for learning, but it’s not very strong as a portfolio project unless you add at least one of these:

Local storage (save daily logs)

Food search

Weekly calorie chart

User login

API food database

Without this, it looks like a simple class assignment, not a real product.
